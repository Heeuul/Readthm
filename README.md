# Readthm

- Checkbox to include type of note inside 
- Answers are given 5 closest answer, with upper and lower limit are randomised, then scrambled 

Note Categories: 
1. Alphabet: 
    - Treble clef 
    - Bass clef 
    - Sharps 
    - Flats 
2. Count: 
    - Semibreve 
    - Minim 
    - Crotchet 
    - Quaver 
    - Semiquaver
